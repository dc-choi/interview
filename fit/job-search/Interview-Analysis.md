# 면접 탈락 원인 분석

## 분석 일자: 2026.04.10

## 개요

- **실질 구직 기간**: 약 15개월 (25년 초 ~ 현재, 이력서가 정상화된 시점 기준)
- 1차 면접 전적: 통과 10 / 탈락 12 / 포기 5 (통과율 ~45%, 포기 제외)
- 2차 면접 전적: **6전 0승 6패** (+ 액션파워 결과 대기)
- 1차·2차 통틀어 **"준비 부족"으로 떨어진 건이 5건** — 가장 명확한 개선 포인트
- 2차 통과율 0%는 구조적 약점이 아니라 **모수 부족**이 핵심 원인

### 1차 면접 핵심 포인트

- 통과율 45%는 나쁘지 않음 — **면접 실력 자체는 문제 아님**
- 전체 서류 통과율은 ~9%로 보이지만, **이력서를 본격적으로 잘 쓰기 시작한 건 25년 초(비누커머스 합격 시점)부터**
- 24년 말 이력서가 부족했던 시기의 지원 건이 전체 수치를 끌어내리고 있어, 실제 서류 통과율은 이보다 높음
- 포기 5건(스마일드래곤, 군돌이, 구름, 코액터스, 테이텀시큐리티)도 모수를 줄이는 요인

## 비누커머스 2차 면접 상세

- **면접 형태**: 컬쳐핏 + 임원(대표, 경영실장, 인사담당자)
- **주요 질문**: 퇴사 사유, 공백기, 지원 동기, 사업적 임팩트, 조직문화, 소통 방식, 커리어 목표
- **기술 질문**: 성능 개선 사례 (Prisma 한방 쿼리 → API 성능 90% 개선), 비개발자 대상 기술 설명
- **특이사항**: 처우 협의까지 진행됨 (연봉, 복지 안내) → 그럼에도 탈락
- **추정 원인**: 기술이 아닌 컬쳐핏/인상 차원의 판단

## 종합 결론

### 1차·2차 통합 — "준비 부족" 탈락 5건

| 단계 | 기업 | 원인 |
|------|------|------|
| 1차 | 두핸즈 | 기술면접 준비 부족 |
| 1차 | 패러다임시프트 | 기술면접 준비 부족 |
| 1차 | 알파코드 | 면접 준비 부족 |
| 2차 | 모니모니 | 준비 부족 |
| 2차 | 호패 | 준비 부족 |

> **면접 실력이 부족한 게 아니라, 준비를 충분히 하고 간 면접은 통과율이 높다.**
> 가끔 준비 없이 가서 떨어지는 패턴이 반복되고 있음.

### 근본 원인

1. **면접 기회(모수) 자체가 부족** — 단, 서류 통과율은 25년 초 이후 개선 중 (24년 말 지원 건이 전체 수치 왜곡)
2. 모수가 적으니 통제 밖 변수(면접관 이슈, 핏 미스매치)에 휘둘림
3. 2차 면접이 구조적으로 약한 게 아니라, 기회가 더 있었으면 통과했을 확률이 높음

## 액션 플랜

1. **파이프라인을 두텁게 유지** — 항상 진행 중인 기업 5곳 이상 확보
2. **면접 기회가 오면 준비를 철저히** — 준비 부족 탈락 5건의 실수 반복 금지
3. **즉석 코테 대비** — 와이즐리·호패처럼 사전 안내 없이 나올 수 있으므로 기본적인 코딩 연습은 유지
4. **이력서는 25년 초부터 개선되어 왔음** — 현재 버전을 기반으로 JD 맞춤 지원에 집중
