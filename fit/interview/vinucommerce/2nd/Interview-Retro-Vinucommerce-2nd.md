# 비누커머스 2차 면접 회고

> **결과**: 탈락 (처우 협의까지 진행 후 컷)
> **회사 관계**: 비누랩스(모회사, 에브리타임 운영)의 계열사. 비누팀 그룹.
> **면접관**: 인사담당자 + 경영실장 + 대표 (3인)

## 진행 흐름

### 1. 자기소개 & 커리어 검증

- **1분 자기소개**
- **첫 회사 퇴사 사유**: SI라 비전 없어 보여서
- **공백기**: 진로 고민 → 42서울에서 동기부여 → 두번째 회사 취업
- **42서울 설명** → 다시 개발자 선택한 이유
- **두번째 회사 퇴사**: "회사 망해서 나왔다"

### 2. 지원 동기

- 서비스 사용 경험, 만족도 → B2C, 커머스 관심
- 커머스 관심 이유: 물류 프로젝트(B2B 뒷단) 경험 → 앞단 B2C 커머스 관심 자연스럽게 확장

### 3. 기술/임팩트 검증

- "기술은 문제 해결 도구라는 문장에 와닿았다. 사업 임팩트 사례?"
- 처음엔 사용성 답변 → 면접관이 매출/비용 관점으로 재질문
- **클라우드 비용 30% 절감 사례** 어필 성공

### 4. 조직문화/소통

- **이전 조직**: 수평적, 직급 없음, 의견 적극 개진
- **이상적 회사**: 공동 목적 단체, 소통으로 개인 needs 해결
- **소통 방식**:
  - 기획자: 의도 먼저 묻고 이해 재확인, 시스템 가능성 제시, 비기술자에겐 기술용어 풀어 설명
  - 프론트엔드: API 문서 최대한 상세 작성

### 5. 자아 인식

- 별명: 똥철, 동물철권 → 친근, 유들유들 이미지
- 본인 자평: "T라서 개발자같다 모먼트 자주 듣는다, 어려워하는 사람도 있다" (본인 인정)
- 대응: "바꿀 수 없으니 그렇구나 하지만, 소통 위해 부드럽게 말하려 노력"

### 6. 회사 비전 매칭

- 비전 4가지(유연한 사고/객관적 판단/과감한 도전/명확한 소통) 중 선택
- **객관적 판단** 선택 → 클라우드 비용 사례 재사용

### 7. 대표 질문 (짧음)

- "성능 개선 사례?" → **Prisma 한방 쿼리 API 90% 개선**
- "스택을 옆 두 분(인사, 경영실장)께 설명" → 개념, 예시, 비유 → **빡집중**

### 8. 마무리 (인사담당자)

- "5년 후?" → 8년차, 개발 리드
- "커리어 최종 목표?" → CTO
- "결과 언제?" → 1~2주
- "왜 물어봤어?" → **합격 시 인수인계 이야기**

### 9. 처우 협의

- 대표, 경영실장 퇴장, 인사담당자 단독
- **복지**: 시차출퇴근, 재택근무, 점심식대
- **연봉**: 회사가 일방적으로 제시 (현 연봉 정확히 + **20% 인상**)

## ✅ 잘 작동한 것 (다음에도 재현할 카드)

1. **B2C, 커머스, 물류 도메인 연결** — 지원동기에서 물류 백엔드 경험 → 앞단 B2C 커머스 관심으로 자연스럽게 확장
2. **클라우드 비용 30% 절감** — 사업 임팩트 사례로 매우 강력. 객관적 판단 비전 매칭에도 재사용
3. **Prisma 90% 개선** — 대표 관심 끌어낸 기술 깊이
4. **스택 비유 설명** — 인사, 경영실장 빡집중 (비기술자에게 전달력 입증)
5. **소통 답변 구체성** — 기획자/프론트엔드 맞춤 응대 패턴 명확

## ⚠️ 다듬을 점 (탈락 가설)

처우 협의까지 갔는데 탈락 = **기술 통과, 미세 컬처핏 또는 정원/예산 컷 가능성 높음**. 가장 의심되는 4가지:

### 1. 잦은 이직 + 외부 귀책 톤 (가장 가능성 높음)

- **첫 회사**: "SI라 비전 없어서" → 회사 탓
- **두번째 회사**: "회사 망해서 나왔다" → 외부 환경
- **공백기**: 진로 고민
- **시사점**: [[My-Motivation-Reasons|FIT 원칙]]에 따르면 외부 귀책은 장기근속 시그널 약화. **윤회 1차 전화 회고와 같은 패턴** — 자기 귀책 + 학습 + 현재 적용으로 재작성 필요

### 2. "T라서 어려워하는 사람 있다" 본인 노출

- 약점 솔직 인정은 OK이지만, **해법이 약함** ("그렇구나 함" + "부드럽게 말하려 노력")
- 컬처핏 면접에서는 **약점을 실제 행동 변화 사례로** 메우는 게 강함
- 다음엔: 어려워하던 동료와 풀어낸 구체적 일화 1개 준비

### 3. "결과 언제? → 인수인계 이야기" 답변

- 조급함 또는 다른 옵션 부재로 비칠 가능성
- **중립 표현**: "다른 회사 일정 조율이 있어서 시점만 확인" 또는 "감사합니다, 다음 단계 안내 일정만 알고 싶었습니다"

### 4. 처우 협상에서 본인 카드 부재

- 회사가 "20% 인상" 일방 제시 → 본인 협상, 역질문 없음
- 컬처핏 평가에서 "본인의 시장가/기대치 분명함"이 가산 요소일 수 있음
- **단**: 신입 협상이 어려운 자리에서 수용은 합리적. 사유 가능성으로는 낮음

## 비누랩스(에브리타임) 재시도 시사점

**같은 비누팀이라 컬처핏, 평가 기준 공유 가능성 高**.

### 긍정 신호 (재시도 의미)

- 비누커머스 **1차 통과 + 처우 협의까지** = 페르소나가 그 그룹에 본질적으로 적합
- 기술적으로 통과한 라인 (Prisma, 클라우드 비용, 스택 설명 다 먹힘)
- 비누랩스(에브리타임) ↔ 비누커머스(에브리유니즈)는 **도메인이 다름** → 평가 기준 부분 다를 가능성

### 부정 신호 (재시도 부담)

- **5년+ 책임급 hard cut** (본인 3년 10개월)
- **컬처핏 평가 데이터 공유 가능성** → 위 4개 우려점이 그대로 영향
- 모회사, 계열사 그룹 채용 데이터베이스 공유는 일반적

### 재시도 시 필수 fork

1. **이직 사유 자기 귀책 + 학습 + 적용 구조로 재작성**
   - 시솔지주 → "운영을 직접 책임지는 환경이 필요해서" (윤회 회고 다듬은 버전 그대로 활용)
   - 트라이포드랩 → "본 계약, 다고객, 표준화 도메인으로 가는 학습 구간"
2. **"T 어려워하는 사람" → 동료와 풀어낸 일화 1개 준비**
3. **5년+ 컷 대응**: 책임급 페르소나 어필 (트라이포드 단독 운영, 기술 결정, 온보딩 사례)
4. **처우 협의 단계 갔을 때 본인 카드 1개**: 시차출퇴근/재택/사이닝/옵션 중 1개 우선순위 명확하게

## 결정 (현재)

- 비누랩스: **보류 유지**. 5년+ 컷 + 계열사 평가 데이터 공유 가능성
- 윤회, 스페이스맵, 토스 결과, 일정 정리 후 재평가
- 재시도 시 위 4가지 fork 필수
