# 글로벌 IT 기업 인터뷰 준비

해외 IT 기업(영어 면접)·글로벌 본사·외국계 지사 인터뷰는 한국 로컬과 세 가지가 다르다. **(1) 서류 탈락률이 높음**(98% 초기 탈락 통계), **(2) JD 일치도와 컬처핏 검증이 집요함**, **(3) "마지막 질문 있나요"의 비중이 절반**. 답변 방어(Defense)만 잘해선 합격하기 어렵고, **능동적 질문(Offense)으로 주도권을 잡아야** 한다.

## 단계별 준비 지도

| 단계 | 핵심 |
|---|---|
| **평상시 준비** | 자기 인식 · 지속 학습 · 네트워킹 |
| **지원 전** | 목표 10개 회사 리스트업 · SNS 정비 · 리쿠르터 관계 |
| **서류 단계** | 이력서 금기/필수 · 브랜드 스토리 일관성 |
| **인터뷰 실전** | 방어(답변) · 공격(질문) 두 축 |
| **후반** | 정식 오퍼 받기 전 퇴사·서명 금지 |

## 1. 평상시 준비 (Career Preparation)

평생 직장이 없다는 전제에서, "준비된 상태"를 기본값으로 둔다.

- **Self-Awareness** — 강점·약점을 냉정히 평가, 명확한 경력 목표
- **Lifelong Learning** — 70(도전적 업무) / 20(동료 학습) / 10(정식 교육) 원칙
- **Networking** — 의미 있는 관계를 **평상시에** 구축. 구직 시작 후 쌓으려면 늦음
- 이력서는 **정기 업데이트** — 반기·분기 단위로 최근 성과 반영

## 2. 지원 전 전략

### 명확한 목표 설정

- **최소 10개 회사** 리스트업, 각 JD·R&R 숙지
- "아무 데나 가겠다"는 태도는 진정성 부족으로 탈락 사유
- 회사 모델·컬처와 본인 추구 방향의 **적합도 검증**

### 소셜 미디어 관리

- LinkedIn·Twitter·GitHub·블로그의 **일관성 있는 본인 브랜드**
- 기업이 지원자 온라인 활동을 상시 모니터링
- 부적절한 콘텐츠·과도한 정치 발언·공백 활동 모두 리스크

### 리쿠르터 관계

- 리쿠르터를 **VIP로 대우**하고 장기 관계 유지
- 추천·내부 기회는 리쿠르터의 신뢰로부터 옴
- 과거 합격 여부와 무관하게 연락 유지

## 3. 서류 단계: 이력서·자소서 금기/필수

### 금지 (Don't)

1. **거짓말·과장** — 장기적으로 반드시 부메랑. 검증 후 꼬리표
2. **창의성 불필요 포지션에 실험적 포맷** — 표준 양식 사용
3. **유행어·전문용어 과다** — 피로감, 실력 회의
4. **경력 단절 미설명** — 의심 요소로 남음
5. **빈약한 스토리** — 경력 나열만 있고 일관성 없음

### 필수 (Do)

1. **브랜드 스토리** — 커리어 전반을 관통하는 본인 관점
2. **명확·가독성** — 한눈에 읽히는 구조
3. **경력 공백 설명** — 공백 사유를 전략적으로 서술
4. **전문가 리뷰** — 채용 경험 많은 사람의 피드백 (중요)
5. **추천서·LinkedIn 최신화**

## 4. 인터뷰 실전: 방어 (Defense)

점수를 잃지 않기 위한 답변 전략.

### 핵심 원칙

- **JD를 완전히 이해**하고 그 언어로 답변
- 모호한 답("매일 다르다") 대신 **분석적·구체적 사례**
- 문제 지적보다 **"어떻게 해결했는가"** 에 집중
- 회사에 대한 깊은 관심 표현 — 창업자 히스토리·최근 분기 실적·최신 제품 방향

### 자주 나오는 답변 함정

| 질문 | 위험 | 안전 |
|---|---|---|
| "왜 이 회사?" | "성장 가능성이 커서" (상투적) | 제품 기능·미션·문화를 구체 언급 |
| "약점은?" | 위장된 장점(완벽주의) | 진짜 약점 + 극복 중인 구체 방법 |
| "이전 상사와 갈등?" | 비난 일색 | 상황 분석 + 본인의 행동 + 학습 |
| "왜 이직?" | 현 직장 디스 | 커리어 방향성 중심 |

### 준비 방법

- **스크립트 작성** → **원어민/경험자 리뷰** → **암기·리허설** → **녹음 확인** → **실전**
- 모의 인터뷰를 **최소 5회** 이상

## 5. 인터뷰 실전: 공격 (Offense)

"마지막 질문 있나요?"는 **마지막 어필 기회**다. 급여·휴가·복지는 나중에. 여기선 **전략적 질문**.

### 효과적 질문의 3조건

- 명확·투명
- 양쪽 모두에게 이득 — 나도 정보 얻고 면접관도 고민하게
- 포지션·조직에 대한 **실제 정보 획득**

### 추천 질문 5가지

1. **"이 직무의 성공 기준은 무엇인가요?"** — 6개월·1년 후 평가 항목
2. **"평가 시스템이 어떻게 구성되나요?"** — 조직 문화·커리어 경로
3. **"지금까지 인터뷰한 지원자 중 본인의 약점으로 보이는 부분은?"** — 즉시 반박할 기회 확보
4. **"Hard skills vs Soft skills 중 이 역할은 어느 쪽이 더 중요한가요?"** — 정확한 역할 파악
5. **"이 포지션이 신설되었나요, 대체인가요? 이유는?"** — 조직 맥락

### 금지 질문

- 급여·휴가·원격근무 비율 (최종 협상 단계에서)
- 회사 홈페이지에 있는 정보
- 너무 사소한 실무(로컬 팀장에게 물어볼 것)

## 6. 인터뷰 당일 주의사항

- **거짓말·과장 금지** (재강조) — 후속 인터뷰·레퍼런스 체크에서 드러남
- **음식·음료 반입 금지** — 옷에 흘리면 주의 산만
- **지각·복장** — 기본 실수로 전체 무효화 가능
- **긍정적 태도** — 심리·감정 요소가 객관적 자격만큼 중요

## 7. 합격 후: 정식 오퍼까지 대기

- 구두 합격은 **오퍼가 아님**
- **공식 Offer Letter + 서명** 전까지 현 직장에 퇴사 통보 금지
- 조건 협상은 Offer Letter 도착 후, 이메일 증빙으로

## 한국 로컬 면접과의 차이

| 축 | 한국 로컬 | 글로벌 IT |
|---|---|---|
| 서류 필터링 | 이력서 + 1차 기술 | ATS + JD 키워드 매칭 강함 |
| 인터뷰 회차 | 2~3회 | 4~7회 (스크리너·TC·행동·온사이트·Bar raiser) |
| 마지막 질문 | 짧게 형식적 | **30% 이상 비중**, 공격의 기술 필수 |
| 평가 기준 | 실무 적합도 중심 | STAR·Leadership Principles·Values 명시 |
| 문화 적합성 | 암묵적 | 명시적 (Culture Fit 단계) |

## 한국 지원자가 자주 저지르는 실수

- 답변이 **수동적**(질문만 답하고 끝) — 적극적 인터뷰 참여 기대치 충족 안 됨
- **스스로 물어본 질문의 깊이 부족** — "없습니다"라고 하면 관심 없음으로 간주
- 영어 답변을 외워 말해 **대화 흐름 깨짐** — 키포인트만 암기, 연결은 즉흥
- 네트워킹·리쿠르터를 **일회성 수단**으로 활용 — 장기 관계로 전환 필요

## 면접 체크포인트 (자가 진단)

- 10개 이상 지원 회사 리스트와 각 JD를 **3줄 요약**할 수 있는가
- 브랜드 스토리를 1분 안에 말할 수 있는가
- 각 전 직장에서 **수치로 표현된 성과 3개**를 가지고 있는가
- 방어 답변 20개·공격 질문 5개를 **원어민 리뷰 받은 스크립트**로 준비했는가
- LinkedIn이 최근 3개월 이내 업데이트 상태인가
