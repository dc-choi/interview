# 윤회 처우 협상 시나리오 (오퍼 단계)

> 전제: **선택지가 좁아 사실상 가야 하는 상황.** 그래도 "현금 얼마"가 아니라 **TC(현금+옵션+사이닝) + 리드 타이틀 + 도메인 = 궤적**으로 받아내는 협상은 본인이 컨트롤한다. 급함을 테이블에서 드러내지 않는 게 1번 원칙.
>
> ⚠️ **최신(2026-05-27 2차 통화): 밴드 4후반~5초 하드캡 확정, 현금 협상 종료. 본인 권고사직 통보 → 무조건 이동.** 핵심은 이제 현금이 아니라 **고용승계 서면화 + 6개월 후 재협상 경로 잠그기**. 아래 "★★ 2차 통화" 섹션이 최신 상태.

## ★ 채용담당 통화 (2026-05-26, 경영지원 팀장) — 확인된 사실

**상황**: 면접(개발팀장+대표) 완료 → 합류 의사, 조건 사전 타진 전화. 서류 제출은 아직(조율 단계). 대표와 조율 후 재연락 예정.

|---|---|---|

### 통화가 바꾼 판단 4가지

1. **"선택지 없어 끌려간다"는 절반만 맞다.** 회사도 **시간 압박**(메인 포지션 공백 임박, 7/1 인수인계 희망) → **사용자 협상력이 생각보다 높다.** 급함을 노출하지 말 것.
2. **평균연봉 3,262만 = 저연봉 시그널이 아니라 사업 구조의 산물.** 서울시가 ~3천 직접 지급하니 회사 명부상 급여가 낮게 잡힘. 회고의 "앵커로 쓰지 말 것"이 옳았고, 이제 이유가 명확.
3. **회사 실부담은 희망연봉 전액이 아님.** 6천 초반 합의해도 **서울시 3천 빼면 회사 보전은 ~3천 + α**. → 협상 카드: "회사 실부담은 절반 수준이니 6천 초반은 무리가 아니다."
4. **미래청년일자리 = 기간 한정 사업 리스크.** 보조금 지원 기간이 끝나면 회사가 **전액 부담**으로 전환. **그 후 연봉 유지, 정규직 전환 보장**을 반드시 확인 (아래 추가 확인 항목).

### 통화 후 즉시 확인/대응 항목

- [ ] **미래청년일자리 지원 기간 + 종료 후 처우**: 보조금 끝나면 6천 초반을 회사가 전액 유지하나? 서면 명시 요구
- [ ] **4대보험 상실 = 서면 오퍼 전 퇴사 금지.** 구두 결정으로 현 직장 퇴사 X. **서면 오퍼 확정 후** 상실신고, 퇴사 (→ [[Salary-Negotiation-Guide]] "서명 전 퇴사 통보 금지")
- [ ] **서울시 거주 요건**: 전입신고 시점, 이사 비용. 입사 전 필요 여부
- [ ] **급여 지급 주체 이원화**(서울시+회사 보전)의 **고용형태, 4대보험, 퇴직금** 처리 명확히
- [ ] 대표 조율 후 재연락 시 → **즉답 금지, 서면 조건 요청**

## ★★ 2차 통화 (2026-05-27, 경영지원 팀장) — 대표 조율 결과 = 현금 협상 사실상 종료

> 결론: **윤회 밴드는 4후반~5초 하드캡. 6천 초반은 죽었다.** 현금 싸움 끝, **구조 리스크 제거로 전환.** (+ 본인 상황 = **권고사직 통보 → 무조건 이동**. 착지 확보가 최우선)

| 항목 | 내용 |
|---|---|
| **포지션** | **백엔드 리드 수준 확정** (백엔드 동료(합류 예정/사수)가 개발 전체, 본인은 백엔드 특화 리드) |
| **밴드** | **4천 후반 ~ 5천 초반 하드캡.** "5천 초과는 어렵다" 명시 |
| **제안** | 현재 연봉(4,560)에서 **5~10% 인상 = ~5,000** → 반올림 5,000대 가능 |
| **경쟁 후보** | 비슷한 경력 다른 후보 있으나 **개발팀장이 본인 선호** → 본인에게 조율 우선권(약한 레버 존재) |
| **고용 구조** | **~2026.12까지 미래청년일자리(서울시 소속, 건강보험상 서울시)** → **2027부터 고용승계로 윤회 직접 채용** |
| **경력 처리** | 6개월간 4대보험상 서울시 소속이나 **경력증명서는 윤회가 발급 가능** |
| **6개월 후** | **12월 종료 시 퍼포먼스 기준 재협상** (윤회 통상 12개월 주기인데 본인은 첫 협상을 6개월 단위로) |
| **고용승계 보장** | **구두로만** "웬만하면 승계, 6개월만 쓸 마인드 아니다, 쭉 같이할 사람 뽑는 것" — **서면 X** |
| **본인 카운터** | **5,100 요청** → 경영지원 팀장 "대표와 논의 후 회신" (대기 중) |

### 판단 — 현금은 끝, 구조를 잠가라

- **5,000~5,100 나오면 수락.** 밴드 천장이라 더 밀면 손실(무조건 이동 상황에 분위기만 상함). 6천은 미련 X.
- **남은 레버 전부 "구조 리스크 제거"에 투입.** 진짜 위험은 숫자가 아니라 고용승계, 근속 산정에 있음.

### 🔴 지금 잠가야 할 4가지 (현금보다 중요)

1. **고용승계 서면화 (최우선)** — "웬만하면"은 구두. **"2026.12 고용승계를 오퍼/계약서에 명시"** 요구. 12월 미승계 = 권고사직→6개월 일경험→재실직(최악) 방지. 거부 시 그 자체가 시그널.
2. **6개월 후 협상의 실질** — 12월에 또 5천 초 밴드면 무의미. **"고용승계 후엔 미래청년일자리 밴드가 아니라 윤회 정직원 밴드로 재논의되냐"** 확인 (서울시 3천 지원 소멸 → 윤회 전액 부담 = 시장가 재논의 여지).
3. **서울시 소속 6개월의 근속 영향** — **4대보험, 퇴직금 기산일, 연차가 2027 승계 시점부터인지** 확인. 6개월 근속 미산입이면 손해.
4. **권고사직 / 4대보험 / 노무사 (미확인)** — 상실신고 서두르기 전 노무사에게 **"권고사직 위로금, 실업급여 권리 포기 없이 정리하는 법"** 확인. 무조건 이동이어도 현 회사 위로금까지 날릴 이유 X.

### 궤적 프레임 (수락 정당화)

권고사직 상황 **공백 없는 착지**(자산) + **리드 타이틀, DPP 규제 도메인, 0→1**(이력서 금광) + **6개월 = 유급 트라이얼**. 5천 초 진입 → 1~2년 리드 → 윤회 재협상 or 키노라이츠급 6천+ 점프. **정답: 들어가는 숫자가 아니라 올라가는 경로를 서면으로 잠근다.**

## 2. 처우 단계 직접 확인 4축 (1차 회고 이월분)

오퍼, 처우 자리에서 **반드시** 정리. 이게 협상 근거이자 입사 리스크 판단 기준.

1. **런웨이, Series A 목표액, 클로징 시점** — 그 연봉을 몇 년 줄 수 있나
2. **118억 R&D 중 윤회 배분액, 연차별 집행, 인건비 충당 여부** — 매출 무관 재원이 되나 (★ 가장 중요)
3. **최근 1~2년 퇴사자 수, 사유** — "9명 퇴사"는 출처 미검증, 직접 확인
4. **동일 직군 연봉 밴드 + 스톡옵션** — 베이스 상한 + 옵션 비중

## 3. 시나리오 — 현금이 5.5 미달로 들어왔을 때

### Step 1. 즉답 금지
- "좋은 제안 감사합니다. **검토 후 답변**드리겠습니다." (며칠 확보)
- 급함, 선택지 없음 **절대 노출 X**. 침묵이 힘.

### Step 2. Base 끌어올리기 (1순위)
- 레버: 위 1번 멘트(R&D, Series A) + 2번 시장선(Node.js 리드 5후반~6천)
- 카운터는 **2~3포인트만**: Base + (사이닝 or 옵션) 집중
- 논리: "직전 대비 인상률 + 리드 책임 범위 + 시장 밴드" (선택지 없음을 논리로 쓰지 말 것)

### Step 3. Base가 5.5에서 막히면 → TC로 갭 메우기
- **사이닝 보너스**: 첫 1~2년 부족분을 일시금으로. 현금 흐름 부담 적어 회사도 수용 쉬움
- **스톡옵션**: 미상장 5년차 = 지분 보상 협상 여지. 단 아래 4번으로 가치 검증
- **TC = Base + 사이닝 + 옵션 기대값**으로 환산해 "실질 5.5 이상"인지 계산

### Step 4. 궤적 자산 확정 (현금 양보 시 반대급부)
현금이 끝내 미달이면 **다음 점프의 밴드를 올려줄 자산**을 확실히 받아낸다:
- **리드 타이틀이 진짜로 붙는가** (CTO 별도 + 본인이 백엔드 리드 — 1차에서 확인된 그림)
- **DPP, ESPR 규제 도메인 + 표준화 테이블 참여** = 이력서 금광 (다음 이직 6~7천대 점프 근거)
- 4천 중반 → (윤회 리드 2~3년) → 다음 이직 점프, 가 성립하면 단기 저연봉은 **궤적 베팅**으로 정당화

## 4. 스톡옵션 받을 때 검증 (형식뿐인 옵션 거르기)

- **Strike Price, Vesting 기간, Cliff** 구조 명확히
- **Dilution**: Pre-A 2회 + Series A 진행 → 후속 라운드 희석 계산
- **유동성**: IPO, 세컨더리 가능성 (미상장이라 단기 현금화 X 전제)
- 기대값 = 확률 × 잠재가치. **연 단위 현금이 부족하면 옵션이 커도 생활 불가** → Base 우선

## 5. 경계할 조합 (이건 선택지 없어도 재고)

> 현금 5.5 미달 **+** 옵션 형식뿐 **+** 리드 타이틀 모호 **+** 런웨이 불투명
> = 궤적도 안 올라가는 순손해. 4축 확인 결과 이 조합이면 수락 재고.

반대로 **R&D 인건비 배분 확인 + 의미 있는 옵션 + 진짜 리드 타이틀 + 규제 모멘텀**이면, 5.5 약간 미달이어도 **2~3년 궤적 베팅으로 OK**.
