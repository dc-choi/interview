# 윤회주식회사 1차 전화 회고 (26.05.15)

## 통화 정보

- **일시**: 2026-05-15(금) 20:06, **24분 23초**
- **상대**: 대표 직접 (개발자 베이스 아님)
- **원래 예상**: 10~15분 전화 → 실제 24분 (긍정 시그널)
- **종료 멘트**: "조금 더 대화 나눠야 할 것 같다, 메시지로 연락 + 이후 일정 제안" → **본 미팅 진행 시사**
- **거주지 확인**: 성남 → 성수 지하철 한 번 OK

## 통화 흐름 요약

1. **DPP/순환경제 사전 학습 확인** — 첫 질문이 "도메인 키워드 파악하고 오셨는지". 사전 정리 효과 큼
2. **물류, VMI 도메인 소개** — 식품(원래 시작) → 제약, 부자재 확장. 대표가 "어쩌다 물류로 접근하게 됐냐"까지 파고듦 = 관심 신호
3. **보안 백그라운드** — 보안학과, SI 솔루션엔지니어, FIDO/패스키. 대표가 ISMS-P 등 인증 경험 물어봄 (보안 중요 시그널)
4. **짧은 근무 이력 사유** — 이썸테크(SI틱→트러블슈팅 경험 부족), 시솔지주(회사 사정), 트라이포드랩(스케일업 갈증)
5. **개발자 커리어 방향** — 프로덕트 엔지니어 동의, 백엔드 주력+프론트 진입 장벽 낮아짐
6. **갈등, 소통** — 환불 정책 일주일 미룬 사례. 리스크 우선 사고
7. **AI 활용도** — Claude Code 하네스, 컨텍스트 엔지니어링, 자가 리뷰. 백엔드 거의 본인이 구축
8. **회사 정보 청취** — 매출 구조(POC→대기업 계약), BM 다층(솔루션+중고의류+자체브랜드+MD), 개발팀 구성, 포지션 확정
9. **입사 가능 시점** — 합격 후 3주
10. **마무리** — 본 미팅 일정 제안 예정

## ✅ 잘 작동한 것

- **도메인 사전 학습** — DPP/순환경제/생애주기 키워드 자연스럽게 첫 답변에 녹임. 대표가 직접 효과 언급 ("키워드 파악해오시면 대화 순조롭다")
- **물류, VMI 도메인 매치** — 대표가 깊게 파고들 만큼 정확한 어필
- **보안 카드 회수** — ISMS-P 경험 없음을 인정하되 "스타트업 인프라 만지며 보안 신경" + 보안학과, FIDO로 갭 메움
- **프로덕트 엔지니어 방향성** — 대표 기조("A부터 Z까지")와 답변 정확히 일치
- **클로드 하네스, AI 협업 파이프라인** — 대표가 "아직 적용 안 함, 솔루션 안정 후 채울 계획"이라 답함 = **사용자 강점 공간이 분명히 있다**는 뜻
- **환불 정책 사례** — 갈등, 소통 질문을 리스크 우선 사고 일화로 받음
- **입사 가능 시점** — 즉답으로 명확성

## ⚠️ 다듬을 점

- **시솔지주 이직 사유 — 외부 귀책 톤**: "회사 사정상 조금 더 안 좋아졌다, 다들 열심히 하셨지만 부득이하게…" → FIT 원칙(자기 귀책 + 학습 + 현재 적용)에 어긋남. **본 미팅에서 같은 질문 재출제 시 다듬어 답변할 것**
- **24분 통화** — 본 미팅은 더 길고 깊을 것. 답변 톤, 구체성 한 번 더 점검 필요

## 본 미팅 준비 — 다음 단계

### 다듬을 답변
1. **시솔지주 이직 사유** — 외부 귀책 → 자기 귀책 + 학습 + 현재 적용으로 재작성
   - 예: "회사 상황도 어려워졌지만, 그때 *제가* 직접 운영을 더 깊게 책임지는 환경이 필요하다고 판단했고, 그래서 다음 회사를 0→1 + 운영까지 가능한 곳으로 골랐습니다"
2. **AI 파이프라인 도입 제안** — "입사 후 6개월 안에 채우겠다" 구체화: 계층형 CLAUDE.md, Subagent, Stop Hook 자가 리뷰, 실DB 통합테스트

### 본 미팅 역질문 강화 (재무, 인력 리스크 검증 — 가장 중요)
- **"올해 대기업, 중견 계약 진행 중이라 하셨는데, 클로징 타임라인과 규모가 어떻게 되나요?"** — 매출 가시성 확인
- **"Series-A 클로징 시점과 규모는 어느 정도 보고 계신가요?"** — 런웨이 확인
- **"개발팀 3명 중 백엔드 충원 사이클이 자주 있었나요? 이전 백엔드 분이 어떤 사유로 퇴직하셨나요?"** — 턴오버 사유 직접 확인 (말로는 자연스럽게)
- **"백엔드 리드라는 포지션의 연봉 밴드는 어느 정도로 보고 계신가요?"** — 평균연봉 3.2천 vs 시장가 갭 확인. 사이닝/옵션 카드 여지 탐색
- **"CTO를 별도로 모실 계획이라 하셨는데 그분이 합류한 후 백엔드 리드의 R&R은 어떻게 정리되나요?"** — 미래 거버넌스

### 어필 카드 (이번 통화에서 먹힌 4개)
1. 보안 (FIDO, 보안학과)
2. 물류, VMI 도메인
3. 자동화 리포트 (IoT 집계 → 알림톡/이메일)
4. 클로드 하네스, AI 협업 파이프라인 (← **입사 후 6개월 책임 영역으로 직접 제안 가능**)

### 결정 트리
- **회사 fit**: 도메인, 기술 매치는 높음. 보안+물류+자동화 카드 잘 먹힘
- **재무 fit**: 위험. 본 미팅에서 매출 가시성, 런웨이, 연봉 밴드, 턴오버 사유 4축 확인 필수
- **합격하더라도 처우 협의 단계에서 시장가, 사이닝, 옵션 비중 따져야 함**. "이 회사에 가고 싶다"가 곧 "어떤 조건이든 OK"는 아님
