# 면접 회고 템플릿

> 면접 후 **24시간 이내 작성**. 회사별 폴더에 `Interview-Retro-{회사명}-{차수}.md`로 저장.
> 핵심: **마스터 역반영** — 새 답변·실제 받은 질문이 마스터에 누락이면 회고 끝에서 마스터 업데이트.

## 면접 정보

| 항목 | 내용 |
|---|---|
| 회사·포지션 | {회사명} {포지션} {차수} (예: "윤회 백엔드 리드 1차") |
| 일시 | YYYY-MM-DD HH:MM, 소요 시간 약 N분 |
| 면접관 | 직함 (대표/CTO/인사/팀원 등 N명) |
| 형식 | 대면/화상 + 장소 |
| 사전 단계 | (커피챗·1차 전화·서류 등) |
| 결과 | 통과/탈락/대기 (확인 후) |

## 진행 흐름 (실제 받은 질문 순서대로)

> 받은 질문을 그대로 적되, 본인이 답한 핵심도 한 줄씩 기록. **본인 표현 그대로**가 아니라 **답한 핵심 요약**으로.

### 1. {질문 카테고리 — 자기소개/이직 사유/기술/컬처핏 등}

- **Q**: "{받은 질문}"
- **A**: {답한 핵심 한 줄}
- **꼬리 Q**: "{꼬리 질문}" → {답한 핵심}

### 2. ...

(질문마다 반복)

## ✅ 잘 작동한 것 (다음에도 재현할 카드)

> 면접관 반응이 좋았던 답변·사례·표현. 구체적으로.

1. **{카드 이름}** — {왜 잘 먹혔는지·면접관 반응}
2. ...

## ⚠️ 다듬을 점 (탈락/감점 가설)

> 답하면서 어색했던 부분, 면접관 반응이 미지근했던 부분, 끝나고 "이렇게 할 걸" 생각한 부분.

1. **{문제 답변/순간}** — {왜 약했는지·다음에 어떻게}
2. ...

## 마스터 역반영 체크리스트 ⭐

> 면접에서 발견한 새 답변·질문을 마스터에 반영해야 학습이 누적됨. 빠지면 같은 회사 다음 차수 또는 다른 회사에서 같은 실수 반복.

- [ ] **새 받은 질문이 [[My-FIT-Answers]]에 없음** → 답변 본문 추가
- [ ] **새 받은 기술 질문이 [[My-Tech-Cards]]·[[My-Tech-Cards-Extended]]에 없음** → 카드 또는 꼬리 추가
- [ ] **마스터 답변 본문이 면접에서 어색했음** → 다듬어서 마스터 본문 교체 (다음 회사 자동 반영)
- [ ] **새 꼬리 질문 패턴 발견** → Extended 또는 카드별 꼬리에 추가
- [ ] **새 역질문 패턴 발견** → [[My-Reverse-Questions]] 추가
- [ ] **민감 질문 변형 발견** → [[Sensitive-Question-Answers]] 보강
- [ ] **회사별 도메인 매핑 한 줄 보강** → [[My-Self-Intro]] 회사별 매핑 가이드 표 추가
