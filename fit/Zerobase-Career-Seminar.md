# 제로베이스취업설명회(25년상반기)

## 취준생이가장많이하는실수Top2

### 1.정보를어떻게찾을것인가
- 회사 정보를 찾을 때 **전자공시(DART)** 를 활용할 것
- 일정 규모 이상의 회사는 전자공시에 반드시 문서를 올림
- 사업보고서에서 매출, 인력, 사업 방향 등 핵심 정보를 확인 가능

### 2.나를어떻게설명할것인가
- 자소서 항목을 정확히 이해하고 작성해야 함

## 자소서항목이해하기

### 지원동기
**기업의 진짜 고민**: 왜 퇴사할까? 뽑은 지 얼마 안 되고 그만둘까?

→ 답변의 본질: **내가 그만두지 않을 이유**

| 구성 | 내용 |
|------|------|
| 회사가 하는 사업 | + 내가 계속 하고 싶은 이유 |
| 회사가 중요하게 생각하는 일 | + 내가 계속 하고 싶은 이유 |

- 비전, 인재상, 기여하고 싶은 바를 엮어서 작성

### 직무관련성공경험
- "훌륭하게", "노력했다", "열심히 했다" → X
- **무조건 정량적인 숫자로** → O

| 질문 | 정량화 방법 |
|------|-----------|
| 돈으로 환산하면 얼마? | 비용 절감, 매출 기여 |
| 얼마만큼 개선? | Before/After 수치 비교 |
| 다른 사람/팀 대비 얼마나? | 상대적 위치, 순위 |

## 커리어전략

### 현실인식
- 경력 있는 신입, 경력직 수시 채용 시대, 공채 종말
- 신입이라면 경험과 경력이 있어야 함
- 경력도 경험도 없다면 **상향지원 + 하향지원 모두** 해봐야 함
- 하향지원도 생각보다 합격률이 안 나옴

### 하향지원을받아들여야하는이유

**1. 경력/경험을 쌓는 회사**
- 첫 직장이 평생 직장이 아닌 시대
- 하루라도 더 빨리 경험을 쌓는 것이 중요

**2. 멘탈관리**
- 합격하면 어떡해? → 합격하고 안 가면 그만
- 지원과 면접이 쉬워짐
- 마음의 평화

### 핵심메시지
- 커리어는 1~2개 회사가 아닌 **10년, 20년 뒤 어떤 일을 하는지**로 결정
- 10년 사이 4~5개 회사를 거칠 수 있음
- **확률 낮은 대기업에 목숨 걸지 말고, 지금 커리어/경험을 쌓을 것**
