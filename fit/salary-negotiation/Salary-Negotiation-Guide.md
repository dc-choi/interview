# 연봉 협상 가이드

> 출처: Career Repotting Project 연봉협상 가이드북 (정구철), 이직의 정석 뉴스레터, 이공이공 처우 협상 실전 사례

---

## 0. 협상이 성립하기 위한 조건

> 출처 추가: 개발자 취업과 이직 한방에 해결하기

협상 테이블에 오르기 위해서는 **레버리지**가 필요하다. 레버리지 없이 앉으면 협상이 아니라 구걸이 된다.

### 필수 전제 조건
- **최소 두 군데 이상에서 오퍼를 받은 상태** — 한 곳만 붙은 상태에서는 협상력이 거의 없다
- **면접에서 좋은 피드백**을 받았거나 **근거가 충분**해야 함 (JD 매칭도, 기술면접 통과, 레퍼런스 체크 통과 등)
- **무턱대고 "희망액만큼 안 주면 안 간다"**는 태도는 협상 자체가 결렬될 가능성이 높음 — 회사는 "이 사람은 다른 곳을 택할 핑계를 찾고 있다"고 판단

### 추가 레버리지가 되는 것
- **현 직장의 복지/성과급/교육비/기술 커리어 관리**를 지원사와 비교해서 **남는 부분(손해 보는 부분)**이 있으면 그 자체가 좋은 협상 카드
  - 예: "현 직장에서는 연말 성과급이 기본급의 20%가 있었는데, 이번 오퍼에는 성과급 항목이 명시되지 않아서…"
- 단, **거짓말은 금지** — 국민연금/레퍼런스로 확인될 수 있고, 들키면 협상 자체가 끝남

### 협상 없이 수락해야 하는 상황
- 한 곳만 붙었고 그 회사가 **내 마지노선 이상**을 제시했을 때
- 회사 사정이 명확히 어렵고 **향후 재협의 약속**이 신뢰할 만할 때
- 처우보다 **배울 수 있는 환경/커리어 도약**이 훨씬 큰 경우

## 1. 핵심 원칙

### 연봉협상 = 등가교환

- 연봉을 낮춰서 갈 지원자도, 특별한 이유 없이 높은 연봉을 줄 회사도 없다
- **누구도 손해보는 결정을 하지 않는다** — 이것이 연봉협상이 등가교환인 이유
- 협상의 목적은 **관철이 아닌 타협점을 찾는 것**

### 명분이 전부다

- 명분이 없으면 때를 쓰는 것밖에 되지 않는다
- 합리적인 근거가 있으면 회사는 움직인다
- 반대로 명분 없이 "그냥 더 주세요"는 통하지 않는다
- **명분에 따라 숫자가 조정되거나 고정된다**

### 회사도 채용을 원한다

- 경력직 TO는 보통 1명, 채용에는 최소 2달 소요
- 서류 검토, 면접, 협상... 회사도 엄청난 시간과 에너지를 쏟은 것
- 면접 통과 = "이 사람과 같이 일하고 싶다"는 확신을 준 것
- **너무 위축되지 말 것** — 연봉협상은 일방적이지 않다

### 연봉협상은 철저한 논리 싸움

- 본인의 입장에서 합리적, 논리적으로 협상하면 회사에도 좋은 인상을 줌
- 반면 이전 면접 때 이미지를 상쇄시킬 정도로 집착하면, 회사 상황에 따라 채용이 진행되더라도 신규 조직에서 높은 연봉에 대한 기대치를 등에 업고 성과를 내기 어려움

## 2. 시장 연봉 조사 (협상의 선제 조건)

**연봉 협상은 면접장에서 시작하는 게 아니라 지원 전에 시작한다.** 시장 데이터를 모르고 들어가면 감으로 숫자를 부르게 되고, 알고 들어가면 근거로 숫자를 부를 수 있다.

### 반드시 확인할 4가지

| 항목 | 확인 방법 | 왜 필요한가 |
|------|----------|-----------|
| **회사 평균 연봉** | 잡플래닛 국민연금 데이터 (가장 신뢰도 높음), 리멤버 | 내 희망 연봉이 회사 수준 대비 어디쯤인지 파악 |
| **시장 연봉 (직무/연차)** | Glassdoor, 원티드, Paylab, 블라인드 | 동일 직무·연차의 시장 중위값 확인 |
| **동종 업계 초봉** | Blind IT업계 테크직군 초봉 시트, codingmonster | 초봉 x1.3~1.5로 경력 연봉 추정 가능 |
| **회사 규모/투자 단계** | 크런치베이스, 뉴스, 기업 정보 사이트 | 연봉 지불 여력 판단 (Seed vs Series B vs 대기업) |

### 사이트별 특성

| 사이트 | 데이터 출처 | 신뢰도 | 비고 |
|--------|----------|--------|------|
| **잡플래닛 (국민연금)** | 회사가 실제 신고한 금액 | 높음 | 전 직군/전 연차 평균이라 직무별 구분 없음 |
| **잡플래닛 (사용자 입력)** | 개인이 직접 입력 | 보통 | 표본 수가 적으면 왜곡 큼, 로그인/멤버십 필요 |
| **리멤버** | 출처 불명 | 보통 | 회사 개요와 함께 빠르게 확인 가능 |
| **Glassdoor** | 익명 제출 | 보통 | 외국계/글로벌 기업에 강함, 한국 스타트업은 표본 부족할 수 있음 |
| **원티드** | 채용 플랫폼 자체 데이터 | 보통 | 직군별 연차별 필터 가능, 백엔드 특화 데이터 확인 |
| **Paylab** | 글로벌 급여 조사 | 낮음 | 글로벌 데이터 혼합, 한국 시장과 괴리 가능 |
| **Blind 초봉 시트** | 현직자 크라우드소싱 | 높음 | 신입 초봉 기준, 경력은 직접 환산 필요 |
| **codingmonster** | 업계 정리 | 보통 | 회사 규모별 초봉 범위 참고용 |

### 경력 연봉 추정법 (초봉 데이터 활용)

동종 업계 초봉 데이터만 있을 때, 경력 연봉을 추정하는 방법:

| 경력 | 초봉 대비 배수 | 예시 (초봉 4,000만 기준) |
|------|-------------|----------------------|
| 3년차 | x1.1~1.2 | 4,400~4,800만 |
| 5년차 | x1.3~1.5 | 5,200~6,000만 |
| 7년차 | x1.5~1.8 | 6,000~7,200만 |

### 데이터 교차 검증

- **하나의 출처만 믿지 말 것** — 출처마다 편차가 크다
- 최소 3개 이상의 출처를 교차 검증하여 **레인지**를 잡는다
- 여러 출처가 가리키는 공통 구간이 실제 시장 가치에 가장 가깝다

## 3. 협상 전 준비할 것

### 사전 조사 체크리스트

| 항목 | 확인 여부 |
|------|----------|
| 회사 평균 연봉 (국민연금 기준) 확인 | |
| 시장 연봉 (직무/연차) 최소 3개 출처 교차 검증 | |
| 회사 규모/투자 단계 확인 | |
| 채용 절차에서 처우협의가 별도 단계인지 확인 | |
| 마지노선 설정 완료 | |
| 명분 3가지 이상 정리 완료 | |

### 마지노선 설정

- **협상 전에 반드시 마지노선을 정할 것** — 협상 중에 정하면 흔들림
- 마지노선은 절대 상대에게 공개하지 않음
- 마지노선 이하의 제안이 오면 거절할 각오가 되어야 함

### 명분 정리

명분으로 쓸 수 있는 것들:
- 시장 데이터 (국민연금, Glassdoor 등 객관적 수치)
- JD 자격요건과의 매칭도
- 기술면접 통과 사실 (역량 검증 완료)
- 포지션의 역할과 책임 범위
- 동종 업계 유사 포지션 처우 수준

## 4. 협상 우선순위

여러 곳에 합격한 가운데 비교할 때:

> **기본급 > 인센티브, Signing Bonus > 복리후생**

- **기본급**: 시장이 나를 평가하는 가치. 다음 이직 때 기준이 됨. 가장 중요
- **인센티브/사이닝 보너스**: 일회성이지만 총 보상에 영향
- **복리후생**: 회사 사정에 따라 가장 먼저 변동되는 부분

연봉보다 **회사의 성장 전망, 대표, 조직원과의 Fit**을 가장 먼저 고려해야겠지만, 처우 협상 시에는 기본급에 집중.

## 5. 협상 프로세스

### 연봉협상의 시작 = 처음 수치를 제시한 시점

- 서류에 희망연봉을 명기했다면, 그것이 시작
- 면접에서 숫자를 언급했다면, 그 숫자를 넘기 어려움
- **현실성 없는 수치를 무작정 높게 부르면**: 자기평가가 부족한 인재, 기회주의자로 낙인
- **통상적인 인상률**: 5~10% or 10~15% (회사, 산업군별 상이)

### 협상 횟수

- 경험상 **2회 정도 (최초 Offer, Last Offer)**에서 마무리되는 경우가 대부분
- 끝없이 밀고 당기는 것이 아님

### 협상 채널

- 모든 오퍼에 대해서는 **서면으로 남기고**, 전화로 해당 내용을 바탕으로 설명
- 서면이 남아야 오해가 없고, 논리를 정리해서 전달 가능

---

## 6. 연봉 조정이 안 될 때 대안

기본급 인상이 어렵다고 할 때 제시할 수 있는 대안들:

1. **사이닝 보너스** — 일시금으로 보전
2. **스톡옵션** — 스타트업에서 유효
3. **직급 조정** — 직급이 올라가면 다음 연봉 테이블도 달라짐
4. **입사일 배려** — 현 직장에서의 인센티브 수령 후 입사
5. **연봉계약 시점 앞당기기** — 입사 후 조기 연봉 재협의
6. **수습 후 연봉 재협의** — 성과 증명 후 재조정 약속
7. **인센티브/성과 보상 구조 명시** — 명확한 성과급 기준 제시 요청

## 7. 연봉협상 가이드라인 (실전)

### 방법

1. **명분을 명확히 하고, 스스로를 객관화. 마지노선을 정한다**
2. **현재 회사와 지원 회사의 가치 및 득실을 파악한다** (주장할 수 있는 논리들을 모은다)
3. **첫 offer를 받고, 마지노선, 입사 여부에 대해 다시 한번 생각을 정리한다**
4. **입사할 마음이면, 연봉에 일희일비 감정을 드러내기 보다, 합격 및 Offer에 대해서 감사의 마음을 전하고, 본인 입장에서 아쉬운 가치들에 대해서 논리적이고, 정중하게 언급한다**
5. **인사담당자(지원 회사)에게 입사할 의지, 믿음을 주는 것이 중요하다**
6. **모든 오퍼에 대해서는 서면으로 남기고, 전화로 해당 내용을 바탕으로 설명, 협상하는 방안으로 진행한다**
7. **시장 상황, 협상에 능하지 않다면 구체적인 수치를 언급하는 것보다는, 여지를 열어 두는 것을 추천한다**

### 톤 & 태도

> "좋은 기회 주셔서 감사합니다. 정말 함께 하고 싶습니다.
> 다만 이런 부분들을 고려했을 때, 이 정도가 적정하다고 생각하는데요.
> 검토 가능하실까요?"

**합리적으로, 감사하게, 정중하게.**

### 절대 하지 말 것

- ~~"얼마든 맞추겠습니다"~~ → 협상력 제로
- ~~"다른 데서 더 준다고 했어요"~~ → 블러핑은 역효과
- ~~"돈보다 성장이 중요합니다"~~ → 진심이어도 연봉 깎이는 근거가 됨
- ~~감정적으로 실망을 드러내기~~ → 프로페셔널하지 않은 인상
- ~~명분 없이 "그냥 더 주세요"~~ → 때쓰기

## 8. 실전 사례: 이공이공 처우 협상

### 타임라인

| 날짜 | 내용 |
|------|------|
| 2025.12.31 | 회사 → 처우안 제시 (기본연봉 4,760만 + 생활지원금 240만 = 총 5,000만) |
| 2026.01.02 | 나 → 연간 총 보상 최소 6,000만 요구 + 3가지 대안 제시 |
| 2026.01.05 | 회사 → 5,000만이 최대 범위라고 회신 |
| 2026.01.05 | 나 → 조건 충족 어려워 합류 불가 결정 |
| 2026.01.05 | 회사 → 아쉽지만 존중한다고 마무리 |

### 잘한 점

1. **명분을 명확히 제시**: "조직적 영향력·리스크·책임 수준과 현재 제안 처우가 다소 간극이 있다"
2. **구체적 기준 제시**: "연간 총 보상 기준 최소 6,000만 원 이상"
3. **대안 3가지 제시** (연봉 조정 / 인센티브 구조 / 보상 체계 협의)
4. **입사 의지를 함께 전달**: "이번 기회를 굉장히 진지하게 검토하고 있고, 함께 하고자 하는 마음은 변함이 없습니다"
5. **마지노선을 지킴**: 5,000만이 최대라는 회신에 합류 불가 결정

### 배울 점

- 협상 메일은 **감사 → 명분 → 구체적 숫자 → 대안 → 입사 의지** 순서
- 거절할 때도 정중하게: "긍정적인 평가와 함께 좋은 제안을 주셨다는 점 다시 한 번 감사드립니다"
- **서면으로 논리를 정리해서 보내는 것**이 전화보다 효과적

## 9. 면접에서 연봉 질문 대응

### "희망 연봉이 어떻게 되세요?"

짧고 명확하게 숫자를 제시. 근거는 물어볼 때만.

> "시장 데이터와 제 경험을 고려해서 [금액] 정도 희망하고 있습니다."

### "현재(직전) 연봉이 얼마예요?"

솔직하게 답하되, **거기서 멈추지 않고** 바로 희망 연봉으로 전환.

> "직전 연봉은 [금액]입니다. **다만** 이전 회사가 시장 대비 낮은 편이었고, 이번에는 [희망 금액]을 희망하고 있습니다."

- 국민연금 조회로 확인 가능하므로 거짓말/회피는 위험
- **"다만"이 앵커를 옮겨주는 핵심 단어**

### "[금액]은 좀 높은데요" / "예산이 맞지 않을 수도"

바로 깎지 말고, 패키지로 전환.

> "유연하게 논의할 수 있습니다. 연봉 외에 스톡옵션이나 성과급 같은 부분도 함께 고려할 수 있어요."

### "다른 곳도 보고 계세요?"

경쟁 시그널 + 여기가 1순위라는 호감을 동시에.

> "몇 군데 진행 중이지만, [회사명]의 [매력 포인트]가 가장 흥미롭습니다."

## 10. 처우협의 단계 메일 템플릿

```
안녕하세요.
먼저 긍정적인 평가와 함께 좋은 제안을 주셔서 진심으로 감사드립니다.

제안해주신 처우 협의안을 면밀히 검토했고, 제 경험과 역량을 높게 평가해주신 점 역시
매우 감사하게 생각하고 있습니다.

다만, 해당 역할에서 요구되는 [구체적 책임/역할]과 현재 제안 처우가 다소 간극이 있다고
판단하여, 현실적인 균형을 맞추기 위한 논의를 한 번 더 부탁드리고자 합니다.

[명분 1: 시장 데이터 근거]
[명분 2: JD 매칭도 / 역량 근거]
[명분 3: 기술면접 통과 사실]

이를 고려하여 연간 총 보상 기준 [희망 금액] 수준이 적정하다고 판단하고 있습니다.

가능하시다면 아래와 같은 방향 중 하나 혹은 조합으로 협의가 가능할지 검토 부탁드립니다.
- 해당 포지션 기준에 맞춘 연봉 조정
- 역할에 상응하는 명확한 인센티브 / 성과 보상 구조 제시
- 혹은 위와 동등한 가치의 보상 체계 협의

이번 기회를 굉장히 진지하게 검토하고 있고, 함께 하고자 하는 마음은 변함이 없습니다.
긍정적인 논의가 되었으면 좋겠습니다. 검토 후 회신 부탁드립니다.

감사합니다.
```
