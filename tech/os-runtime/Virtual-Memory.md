# Virtual Memory

물리 메모리와 보조저장장치를 추상화하여 프로세스에게 독립적이고 연속적인 주소 공간을 제공하는 체계. 메모리 계층부터 할당 방식, 페이징/세그멘테이션, 페이지 교체 정책, 스레싱까지 다룬다.
