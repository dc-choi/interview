# TS 타입 (types 인덱스)

타입 시스템 개념 노트와 타입별 예제 코드 폴더 모음. 상위 인덱스는 [[타입스크립트(TS)|TS 인덱스]].

## 목차

## 예제 코드 폴더
- `primitive/` — number, string, boolean, literal, null, undefined
- `object/` — 객체와 구조적 타이핑
- `array/` — 배열과 튜플
- `enum/` — enum
- `any-unknown/` — any, unknown
- `void-never/` — void, never
- `type-index/` — 인덱스 시그니처
