# 리더십 역발상 — -10x 리더 피하기, 10x 리더 되기

리더십을 더하기가 아니라 **빼기**로 접근하는 프레임. 팀 효율을 10배로 높이는 리더(10x)가 되려고 애쓰기 전에, 팀 효율을 1/10로 떨어뜨리는 리더(-10x)의 행동부터 제거한다. 무엇을 추구할지보다 **무엇을 피할지 아는 것**이 먼저다.

## 역발상 (Inversion)

Charlie Munger의 사고 도구. 탁월해지는 방법을 직접 찾기보다 실패를 만드는 요인을 뒤집어 제거하는 접근 — 똑똑해지려 하기보다 어리석음을 피하는 힘이 장기적으로 더 크다. 리더십에 적용하면: 존경받는 리더의 조건을 나열하기 전에, 팀원이 떠나게 만드는 리더의 행동 목록을 만들고 자신을 대조한다. 피해야 할 것은 구체적이고 즉시 점검 가능하지만, 추구해야 할 것은 추상적이기 쉽다.

## -10x 특성과 10x 원칙의 대응

-10x의 각 특성을 뒤집으면 10x의 원칙이 된다.

| -10x 리더 (피할 것) | 10x 리더 (실천할 것) |
|---|---|
| 권위주의 — 의견 무시, 독재적 결정 | 권한 부여 — 주인의식과 집단 성장 촉진 |
| 편견과 차별 | 공정성과 포용 — 다양성 존중, 동등한 기회 |
| 소통 부족 — 경청 없음 | 열린 소통 — 투명성, 적극적 경청, 건설적 피드백 |
| 약속 불이행 — 신뢰 상실 | 신뢰성과 책임 — 약속 이행, 성공 공유 |
| 오만 — 실수 부인 | 겸손과 성장 마인드셋 — 학습 개방, 실수 인정 |
| 무능과 정체 — 역량 개발 없음 | 지속적 발전 — 전문성 개발, 지식 공유 |
| 비윤리 — 부정직 | 윤리적 리더십 — 도덕 기준, 투명성 |
| 마이크로매니지먼트 — 자율성 억압 | 신뢰와 위임 — 의미 있는 책임 부여 |
| 공감 부족 — 어려움 무시 | 감성 지능 — 팀원 웰빙에 진정한 관심 |
| 인정 인색 — 동기 저하 | 인정과 동기부여 — 성과 축하, 성장 기회 |

## 실천 순서 — 빼기 먼저, 더하기 나중

1. **-10x 목록으로 자기 점검** — 지금 하고 있는 행동이 왼쪽 열에 있는가. 하나라도 있으면 그것부터 제거. 좋은 원칙 아홉 개를 실천해도 마이크로매니지먼트 하나가 팀 효율을 무너뜨린다
2. **제거 후 대응 원칙 실천** — 뒤집은 오른쪽 열이 실천 항목이 된다
3. **지속 반복** — 한 번의 점검이 아니라 주기적 대조

## 체크포인트

- 리더십 개선을 더하기(덕목 추가)로 접근하는가, 빼기(결함 제거)로 접근하는가
- -10x 목록 중 자신에게 해당하는 행동을 구체적으로 짚을 수 있는가
- 팀 효율을 깎는 행동 하나가 좋은 원칙 여러 개를 상쇄한다는 비대칭을 인식하는가
- 역발상을 리더십 외의 설계 결정에도 적용할 수 있는가 (성공 조건 나열 대신 실패 요인 제거)
