# 생성형 AI 시장 경쟁 구도 (LLM Market Landscape)

생성형 AI 서비스 시장은 단일 사업자 독점에서 소수 과점(3강)으로 재편되고 있다. 모델 성능 자체가 상향 평준화되면서 다음 경쟁 축이 **어디에 닿느냐(유통, Distribution)**로 넘어갔고, 점유율을 읽는 지표가 채널마다 달라 해석에 주의가 필요한 단계다.

## 웹 방문 기준 점유율 (2026.04)

| 서비스 | 웹 점유율 |
|---|---|
| 챗GPT | 54.7% |
| 제미나이(Gemini) | 27.4% |
| 클로드(Claude) | 8.2% |
| 딥시크(DeepSeek) | 4.1% |
| 그록(Grok) | 2.8% |
| 퍼플렉시티(Perplexity) | 1.5% |
| 코파일럿(MS Copilot) | 1.3% |

## 독점에서 과점으로 — 추세

- **챗GPT**: 76.5%(2025.02) → 62.4%(2026.02) → 54.7%(2026.04). 1년여 만에 약 22%p 하락.
- **제미나이**: 5.6% → 27.4%로 급등.
- **클로드**: 월 웹 방문량 2.03억회(2026.01) → 8.24억회(2026.04), 한 분기 만에 약 306% 증가.
- 선두(챗GPT)의 절대 방문량은 6개월간 약 6% 줄며 사실상 정체.

핵심은 선두가 무너진 게 아니라 **시장이 커지는 동안 그 증분(성장분)을 후발 주자가 흡수**하는 구조라는 점이다. 절대값 정체와 점유율 하락이 동시에 일어난다. 점유율 그래프만 보면 급락처럼 보이지만 절대 사용량은 보합인, 전형적인 "성장 시장에서의 상대적 후퇴"다.

## 무엇이 점유율을 가르나 — 유통(Distribution)

성능 격차가 좁혀진 구간에서는 **이미 사용자가 있는 접점에 모델을 얹는 능력**이 점유를 좌우한다.

- **제미나이**: 검색, 모바일 OS(안드로이드), 오피스 스위트(워크스페이스)에 번들. 사용자가 별도 사이트를 방문하지 않아도 기존 동선 위에서 도달한다.
- **클로드**: 개발자, 기업 시장에서 쌓은 입지가 일반 소비자로 확장. B2B 신뢰가 B2C로 전이되는 경로.
- **단말 기본 탑재**: OS 음성 비서가 복수 모델을 선택지로 개방하면, 기본 진입점을 잡은 모델이 디폴트 트래픽을 흡수한다. 경쟁이 모델 품질에서 유통(단말, OS) 영역으로 확장되는 신호.

이는 [[Distribution-As-Moat|도달력이 새 해자]]의 직접 실증이다. 모바일 시대의 해자가 리텐션과 네트워크 효과였다면, 모델이 상향 평준화된 구간의 해자는 **남이 깔아둔 접점(OS, 검색, 오피스)에 올라타는 유통**이다.

## 지표 해석의 함정 — 웹 방문량 ≠ 시장 점유율

웹사이트 방문량은 여러 사용 채널 중 하나일 뿐이다.

- **빠진 것**: 모바일 앱 사용량, 기업용 API 호출량.
- 따라서 웹 점유율이 곧 매출, 실사용 점유율은 아니다. API, 앱 비중이 큰 사업자는 웹 지표에서 구조적으로 과소평가된다.
- 시니어 관점: "점유율 1위" 헤드라인을 볼 때 **무엇을 센 숫자인가(분모와 채널)**를 먼저 확인한다. 같은 시장도 웹, 앱, API, 매출 중 어느 렌즈로 보느냐에 따라 순위가 뒤집힐 수 있다.

## 체크포인트

- 모델 성능이 비슷해지면 다음 경쟁 축은 무엇인가 → 유통(이미 깔린 접점), 가격, 생태계 락인.
- "점유율" 수치를 인용할 때 측정 채널(웹, 앱, API, 매출)을 함께 말하는가.
- 우리 제품의 도달이 자체 트래픽에만 의존하는가, 남이 깔아둔 접점에 올라타는가.
- 선두의 절대 지표가 정체일 때, 성장은 "시장 증분을 누가 흡수하느냐"로 결정된다.

## 사례

- 2026.04 웹 방문 기준 7대 생성형 AI 서비스 점유율 측정(마케팅 분석 업체가 시장조사 업체 자료로 분석): 챗GPT 54.7%, 제미나이 27.4%, 클로드 8.2% 순.
- 2026.06 애플 개발자 회의: 차기 OS 음성 비서(시리)에 챗GPT, 클로드, 제미나이를 사용자가 골라 쓰도록 개방 — 단말 유통 경쟁의 신호.
