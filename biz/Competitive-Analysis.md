# 경쟁사 분석

## Porter's Five Forces

Michael Porter가 제시한 산업의 경쟁 강도와 수익성을 분석하는 프레임워크이다.

1. **기존 경쟁자 간 경쟁 (Rivalry):** 시장 내 기존 플레이어들의 경쟁 강도
2. **신규 진입자 위협 (New Entrants):** 새로운 경쟁자가 시장에 들어올 가능성
3. **대체재 위협 (Substitutes):** 다른 방식으로 같은 문제를 해결하는 제품
4. **공급자 교섭력 (Supplier Power):** 공급자가 가격/조건을 좌우하는 힘
5. **구매자 교섭력 (Buyer Power):** 고객이 가격/조건을 좌우하는 힘

**한계:** 플랫폼, 네트워크 효과, 데이터 기반 시장에서는 설명력이 떨어진다.

## 경쟁 해자 (Competitive Moat)

Warren Buffett이 제시한 개념. 경쟁자의 진입을 막는 구조적 장벽이다.

| 해자 유형 | 설명 | 예시 |
|---|---|---|
| 네트워크 효과 | 사용자가 늘수록 가치 증가 | 카카오톡, 인스타그램 |
| 전환 비용 | 다른 제품으로 바꾸기 어려움 | SAP, 항공 마일리지 |
| 규모의 경제 | 규모가 클수록 단가 하락 | AWS, 쿠팡 |
| 브랜드 | 신뢰와 인지도 | 애플, 토스 |
| 도메인 특화 | 특정 분야의 전문 지식/데이터 | 전례력, 교구 계층 |

## SWOT 분석

내부 역량(S/W)과 외부 환경(O/T)을 2×2 매트릭스로 정리한다.

| | 긍정 | 부정 |
|---|---|---|
| **내부** | Strengths | Weaknesses |
| **외부** | Opportunities | Threats |

Porter's Five Forces(외부)와 SWOT(내부+외부)를 함께 사용하면 더 완전한 분석이 된다.

## 현장 적용: school-manage

- **직접 경쟁자:** 브랜치스(개신교, 95교회/220명) — 유사 기능이지만 도메인 장벽
- **대체재:** 엑셀, 카톡, 수기 — "충분하다고 느끼는" 사용자 존재
- **해자:** 도메인 특화(전례력, 교구 계층, 세례명) — 범용 도구나 개신교 도구의 진입 차단
- **Five Forces:** 신규 진입 위협 낮음(도메인 지식 필요), 대체재 위협 높음(엑셀이 공짜)

## 면접 포인트

Q. 경쟁사를 어떻게 분석했는가?
- Porter's Five Forces로 산업 구조, SWOT으로 내부+외부 분석
- 도메인 특화(전례력, 교구 계층)를 경쟁 해자로 활용
- 직접 경쟁자(브랜치스) 벤치마크 + 대체재(엑셀) 위협 대응
