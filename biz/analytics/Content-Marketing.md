# 콘텐츠 마케팅

## 콘텐츠 마케팅 퍼널

사용자의 구매 여정 단계에 맞는 콘텐츠를 제공한다.

| 단계 | 목표 | 콘텐츠 유형 |
|---|---|---|
| **TOFU (인지)** | "이런 게 있구나" | 블로그, SNS, 카드뉴스, 4컷 만화 |
| **MOFU (고려)** | "나한테 맞을까?" | 사용자 인터뷰, 비교 가이드, 데모 |
| **BOFU (전환)** | "시작해볼까" | 무료 체험, 온보딩 가이드, 성공 사례 |

## 채널 유형

| 유형 | 설명 | 특징 |
|---|---|---|
| **Owned** | 자사 소유 (블로그, 뉴스레터) | 통제 가능, 느린 성장 |
| **Earned** | 자발적 노출 (입소문, PR, 리뷰) | 신뢰도 높음, 통제 어려움 |
| **Paid** | 유료 광고 (구글, SNS) | 즉시 효과, 비용 발생 |

## SEO & Topic Cluster 모델

콘텐츠가 검색엔진에서 발견되려면 SEO 전략이 필요하다.

**Topic Cluster:**
- **Pillar Page:** 핵심 주제의 종합 가이드 (3,000+ 단어)
- **Cluster Content:** 하위 주제별 상세 콘텐츠
- 내부 링크로 연결하여 검색 권위(authority) 구축

**검색 의도(Search Intent) 4유형:**
- Informational: "PMF란 무엇인가" (정보)
- Navigational: "Slack 로그인" (탐색)
- Commercial: "Notion vs Coda 비교" (조사)
- Transactional: "프로젝트 관리 도구 무료 체험" (행동)

## 콘텐츠 ROI 측정

**Attribution(기여도) 모델:**

| 모델 | 설명 | 적합한 경우 |
|---|---|---|
| First-touch | 첫 접점에 100% 기여 | TOFU 효과 측정 |
| Last-touch | 마지막 접점에 100% 기여 | BOFU 효과 측정 |
| Linear | 모든 접점에 균등 배분 | 단순한 공정 평가 |
| Time-decay | 전환에 가까울수록 높은 기여 | B2B 긴 세일즈 사이클 |

**핵심:** 콘텐츠는 **복리(compound) 효과**가 있다. 좋은 글은 수년간 유기 트래픽을 생성하므로 6~12개월 단위로 평가한다.

## Community-Led Content

콘텐츠 → 커뮤니티 유입 → 사용자 생성 콘텐츠(UGC) → 더 많은 유입의 **선순환(Flywheel)**.

**핵심 원칙:**
- Give before you take — 가치를 먼저, 홍보는 나중에
- Champion 육성 — 열정적 사용자를 앰배서더로
- 커뮤니티 크기보다 참여 깊이(engagement)가 중요

## 현장 적용: school-manage

- **3채널 운영:** 인스타그램(TOFU, 인지) + 사제 네트워크(BOFU, 전환) + 입소문(MOFU~BOFU)
- **데이터 기반 축소:** 초기 15개 4컷 만화 + 7개 릴스 → 피드백 후 4개로 축소 (낮은 가시성)
- **최고 효과:** 콘텐츠 마케팅이 아닌 Earned 채널(사제 네트워크, 2일 57계정)이 실제 전환 주도
- **인사이트:** TOFU(인지)와 BOFU(전환)의 최적 채널이 다르다 — 각 단계에 맞는 채널 배치

## 면접 포인트

Q. 콘텐츠 마케팅은 어떻게 했는가?
- 퍼널별 채널 배치: TOFU(인스타) / MOFU(입소문) / BOFU(사제 네트워크)
- 데이터 기반 운영: 콘텐츠 15개 → 4개 축소 (ROI 측정 후)
- 콘텐츠 마케팅의 한계를 인식하고 Earned 채널(커뮤니티)에 집중
