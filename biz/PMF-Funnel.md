# PMF 검증 & 전환 퍼널

## Product-Market Fit이란

Marc Andreessen의 정의: **"사용자가 제품 없이는 안 되는 상태."**

PMF가 있으면 제품이 알아서 팔리고, 없으면 아무리 마케팅해도 허비다. 스타트업의 유일한 목표는 PMF를 찾는 것이다.

## PMF 측정 방법

### Sean Ellis Test (40% 법칙)
"이 제품을 더 이상 사용할 수 없다면 어떤 기분일까요?"
- 매우 실망 / 약간 실망 / 실망하지 않음 / 더 이상 사용하지 않음
- **"매우 실망" 40% 이상이면 PMF 달성**
- 최소 100명 이상의 활성 사용자, 2주+ 사용 경험 필요

### Dan Olsen의 PMF 피라미드
아래에서 위로 쌓아 올린다. 하나라도 빠지면 무너진다.
1. **타겟 고객** — 누구를 위한 것인가?
2. **미충족 니즈** — 어떤 문제가 해결 안 되고 있는가?
3. **가치 제안** — 어떻게 해결하는가?
4. **기능 세트** — 가치를 전달하는 구체적 기능
5. **UX** — 사용자가 가치를 실제로 경험하는 방식

### 리텐션 커브
코호트별 리텐션을 시간축으로 그린다.
- 커브가 0으로 수렴 → PMF 없음
- 커브가 수평으로 안정(flattening) → PMF 신호
- 커브가 상승(smile curve) → 강한 PMF

## Pre-PMF vs Post-PMF 전략

| 영역 | Pre-PMF | Post-PMF |
|---|---|---|
| 목표 | PMF를 찾는 것 | PMF를 확장/방어하는 것 |
| 제품 | MVP로 빠른 가설 검증 | 안정성, 확장성, 기술 부채 해소 |
| 고객 | 극도로 좁은 세그먼트 집중 | 인접 시장 확장 (Crossing the Chasm) |
| 성장 | "Do Things That Don't Scale" (Paul Graham) | 성장 엔진 가동, 유료 채널 |
| 지표 | **리텐션 > 성장률** | **성장률, CAC/LTV, Unit Economics** |
| 마케팅 | 유료 광고 금지, 유기적 유입만 | 유료 + 콘텐츠 + 파트너십 |

**가장 흔한 실패: PMF 전에 스케일링 (Premature Scaling)** — CB Insights 스타트업 실패 1위 원인

## MVP 검증 스펙트럼

비용이 낮은 것부터 순서대로:

| 방법 | 비용 | 설명 |
|---|---|---|
| **문제 인터뷰** | 최소 | 문제가 존재하는지 확인 |
| **Fake Door** | 낮음 | 기능이 있는 것처럼 버튼을 만들고 클릭률 측정 |
| **랜딩 페이지** | 낮음 | 가치 제안 + 이메일 수집으로 수요 측정 |
| **Wizard of Oz** | 중간 | 자동화처럼 보이지만 실제로는 수동 처리 |
| **Concierge** | 중간 | 수동으로 서비스를 제공하며 반응 관찰 |
| **단일 기능 MVP** | 높음 | 핵심 한 가지만 구현한 실제 작동 제품 |

**원칙:** 가장 위험한 가정을 먼저, 가장 싸게 테스트한다.

## 전환 퍼널

사용자가 제품을 알게 된 순간부터 핵심 가치를 경험하기까지의 단계를 정의하고, 각 단계의 전환율을 측정한다.

**TOFU / MOFU / BOFU:**
- TOFU (Top of Funnel): 인지 — 제품을 알게 됨
- MOFU (Middle of Funnel): 고려 — 가입, 탐색
- BOFU (Bottom of Funnel): 전환 — 핵심 행동, 유료 전환

**병목 식별 원칙:** 가장 큰 이탈이 발생하는 단계에 집중 투자한다.

## 현장 적용: school-manage

**퍼널 (누적 전환 39%):**
모임가입 → 그룹생성(88%) → 학생등록(71%) → 출석기록(84%) → WAU(74%)

**병목 해결:**
- 랜딩 부재 → 랜딩 페이지 구현 → 이탈률 81.8% → **12.5%** (69pp 개선)
- 학생 등록 병목(29% 이탈) → Excel 일괄 업로드

**코호트 분석:**
- 2월 중순: 0%, 2월 하순: 15%, 3월: 36% → 리텐션 커브가 상승 추세

**PMF 신호:**
- S유형(팀 사용) WAU 100% 리텐션
- 사제 네트워크 바이럴 (2일 57계정)
- 킬러 피처 발견 (축일 카드 인당 8.5회)

## 면접 포인트

Q. PMF를 어떻게 검증했는가?
- 5단계 퍼널 정의 → 단계별 전환율 추적 → 병목 식별/해결 반복
- 코호트 분석으로 제품 개선 효과를 시간축 검증 (0%→15%→36%)
- Sean Ellis test 기준: S유형 100% 리텐션이 PMF의 강한 신호
